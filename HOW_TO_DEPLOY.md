# 🎟️ Raffle App — Deploy in 5 Minutes

## STEP 1 — Create GitHub account
1. Go to **github.com**
2. Click "Sign up" (free)
3. Verify your email

---

## STEP 2 — Create a new repository
1. Click the **+** icon (top right) → "New repository"
2. Name it: `raffle`
3. Set to **Public**
4. ✅ Check "Add a README file"
5. Click **Create repository**

---

## STEP 3 — Upload all these files to GitHub
1. On your new repo page, click **"Add file"** → "Upload files"
2. Drag and drop the **entire contents** of this folder:
   - `index.html`
   - `package.json`
   - `vite.config.js`
   - `vercel.json`
   - `.gitignore`
   - `src/` folder (contains `main.jsx` and `RaffleApp.jsx`)
3. Click **Commit changes**

---

## STEP 4 — Create Vercel account
1. Go to **vercel.com**
2. Click "Sign Up" → choose **"Continue with GitHub"**
3. Authorize Vercel

---

## STEP 5 — Deploy
1. On Vercel dashboard click **"Add New Project"**
2. Find your `raffle` repo → click **Import**
3. Framework: select **Vite**
4. Leave everything else as default
5. Click **Deploy**

⏱️ Wait ~60 seconds...

6. Vercel gives you a live URL like:
   **`your-raffle.vercel.app`** ← this is your link!
